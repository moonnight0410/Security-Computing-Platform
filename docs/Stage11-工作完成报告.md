# Stage 11 工作完成报告

| 项 | 内容 |

## 2. 本阶段完成项

### 2.1 规则资产复用

- 新增规则模板模型与接口。
- 新增规则条目模型与接口。
- 支持保存当前规则树为模板。
- 支持保存单条规则为复用条目。
- 支持前端加载模板库与条目库。
- 支持模板整套套用。
- 支持条目插入指定规则分组。

### 2.2 组合规则编排

- 新增规则树基础服务 `rule_logic.py`。
- 支持 `group` / `rule` 两类节点。
- 支持嵌套 `AND` / `OR` 分组。
- 支持旧平铺规则自动兼容为根 `AND` 分组。
- 执行引擎改为按规则树求值。
- 规则计数改为递归统计叶子规则数。

### 2.3 任务与输出治理看板

- 新增治理看板汇总服务。
- 新增治理看板接口 `GET /api/governance/dashboard`。
- 前端工作台新增治理看板组件。
- 支持任务状态、输出状态、规则包状态、审计总量、待审结论等汇总展示。
- 支持最近任务、最近输出申请、最近输出文件、最近归档列表。

### 2.4 前端主题升级

- 全站主题改为蓝白配色。
- 背景、按钮、标签、面板、输入框、卡片全部切换到蓝白体系。
- 新增治理看板与规则树专用样式。

### 2.5 前端工作台收口与规则包中心重排

- 删除看板上方的着陆页、说明区和非必要介绍文案，前端首页仅保留工作台业务操作区块。
- 恢复规则包编辑页入口，改为工作台顶部轻量工具栏按钮，避免入口丢失。
- 规则包编辑页增加“返回工作台”入口，形成双向切换。
- 规则包中心不再采用横向三栏并排，改为纵向分段布局，消除拥挤与内容重叠问题。
- 规则包编辑表单改为更稳的双列栅格，在窄屏下自动回落为单列。
- 首页与规则包页文案去除开发说明式、自我评价式和内部边界复述式表达，保留正常产品级界面语言。

### 2.6 检查脚本与版本同步

- 新增 `scripts/run-stage11-checks.ps1`。
- 前端版本号升级为 `0.11.0-stage11`。
- 后端版本号升级为 `0.11.0-stage11-domain-local`。

## 3. 截至当前已实现的全部功能

按要求列出当前项目全部已实现功能。

### 3.1 本域边界与安全约束

- 原始数据只能在本域内导入、落盘、处理。
- 加密后数据、哈希数据、去标识数据同样不得出域。
- 不提供对象级明细输出。
- 不提供原始主键、子键、派生键、去标识摘要的对外输出。
- 仅允许执行回执、结论声明、满足最小阈值的聚合统计等受控输出形式。

### 3.2 数据导入与字段画像

- 支持导入 `CSV`、`XLSX`、`XLS`。
- CSV 支持字段画像分析。
- 支持统计字段名、推断类型、空值数、非空数、重复数、样例值。
- 导入文件仅写入本地 `workspace/imports`。

### 3.3 字段映射与本地状态管理

- 支持主键字段映射。
- 支持子键字段映射。
- 支持敏感字段映射。
- 支持聚合统计分组字段映射。
- 系统本地状态由 SQLite 持久化。
- 保留旧 JSON 状态兼容迁移能力。

### 3.4 本地任务执行

- 支持创建本地任务草稿。
- 支持单数据集本地执行。
- 支持主键完整率、子键完整率、重复组等摘要统计。
- 支持去标识处理参与本地计算但不对外返回。
- 支持规则表达式命中统计。
- 支持执行结果本地留存。

### 3.5 输出策略控制

- 支持 `local_only`。
- 支持 `execution_receipt`。
- 支持 `manual_assertion`。
- 支持 `aggregate_summary`。
- 聚合统计仅允许单维粗粒度分组。
- 聚合统计最小阈值默认 `10`。
- 小样本结果会被抑制处理。

### 3.6 规则表达式能力

- 支持平铺规则条目输入。
- 支持嵌套规则树。
- 支持 `AND` / `OR` 组合分组。
- 支持 `eq`。
- 支持 `neq`。
- 支持 `exists`。
- 支持 `not_empty`。
- 支持 `in`。
- 支持 `gte`。
- 支持 `lte`。
- 输出仅保留通过数、失败数、未知数，不返回命中明细。

### 3.7 规则资产复用

- 支持规则模板创建。
- 支持规则模板列表。
- 支持规则模板整套套用。
- 支持规则条目创建。
- 支持规则条目列表。
- 支持规则条目插入指定分组。

### 3.8 规则包治理主链

- 支持规则包登记。
- 支持规则包单个验签。
- 支持规则包单个审批。
- 支持规则包批量验签。
- 支持规则包批量审批。
- 支持规则包中心独立管理。
- 支持规则包草稿创建。
- 支持规则包编辑。
- 支持规则包修订快照。
- 支持已审批规则包自动派生修订草稿。
- 支持规则包废弃。
- 支持规则包受限删除。

### 3.9 规则包签名与验签

- 支持本域 RSA 公私钥签名与验签。
- 支持受信任签名人清单。
- 支持规则包签名脚本。
- 支持编辑后签名失效。
- 支持草稿无签名保存。
- 支持修订级验签。

### 3.10 规则包修订差异与回溯

- 支持修订字段级差异对比。
- 支持修订结构级变化对比。
- 支持识别新增、删除、变更规则节点。
- 支持修订直接派生关系识别。
- 支持任务对规则包修订的引用回溯。
- 支持当前修订与历史修订引用分离统计。

### 3.11 结论声明与输出审批

- 支持结论声明草稿生成。
- 支持正式审核通过。
- 支持审核驳回。
- 支持执行人与审核人分离。
- 支持创建输出申请。
- 支持审批输出申请。
- 支持输出包预览。
- 支持输出文件本地落盘。

### 3.12 输出文件与归档封存

- 支持输出文件列表。
- 支持多文件归档封存。
- 支持生成归档清单。
- 支持归档签名文件。
- 支持归档验签报告。
- 支持归档报告重新验签。

### 3.13 审计与日志

- 关键动作写入本地审计表。
- 审计 JSONL 同步落盘。
- 审计链包含 `previous_hash` 与 `entry_hash`。
- 支持审计链完整性校验。
- 应用日志落盘至 `workspace/logs`。

### 3.14 前端页面与本地演示

- 支持本域工作台视图。
- 支持规则包中心视图。
- 支持治理看板视图块。
- 支持数据导入、字段映射、任务、结果、审计、输出、归档操作。
- 支持规则包中心中的修订治理工作流。
- 支持修订差异和引用回溯可视化审阅。
- 支持蓝白主题展示。
- 支持工作台顶部轻量入口切换规则包编辑页。
- 支持规则包中心纵向分段布局，避免横向三栏拥挤与重叠。

### 3.15 脚本与自动检查

- 已提供 `run-stage3-checks.ps1` 至 `run-stage11-checks.ps1`。
- 已提供前后端启动脚本。
