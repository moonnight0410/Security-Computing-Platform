# Stage 11 规则模板复用与组合规则编排说明

## 1. 阶段目标

Stage 11 重点完成四项增强：

- 规则模板复用
- 规则条目复用
- 更复杂的 AND / OR 组合规则编排
- 任务与输出治理看板

同时按要求将前端主题统一调整为蓝白配色。

## 2. 规则模板与条目复用

### 2.1 规则模板

新增规则模板库，模板保存的是完整规则表达式结构，不包含任何业务数据。

支持：

- 保存当前规则树为模板
- 查看模板列表
- 将模板整套套用到当前规则包草稿

### 2.2 规则条目

新增规则条目库，条目保存的是单条规则节点。

支持：

- 将当前编辑中的单条规则保存为条目
- 查看条目列表
- 将条目插入到指定规则分组

## 3. AND / OR 组合规则编排

### 3.1 规则树结构

规则不再只支持平铺数组，现支持树形结构：

- `group`
- `rule`

其中 `group` 支持：

- `logic = and`
- `logic = or`
- `children = [...]`

### 3.2 执行语义

- `AND` 组中只要有一条明确失败，则整组失败。
- `AND` 组全部明确通过，则整组通过。
- `AND` 组存在未判定且没有失败，则整组结果为未判定。
- `OR` 组中只要有一条明确通过，则整组通过。
- `OR` 组全部明确失败，则整组失败。
- `OR` 组存在未判定且没有通过，则整组结果为未判定。

### 3.3 向后兼容

旧版本平铺规则数组会自动视为根 `AND` 分组。

## 4. 治理看板

新增治理看板接口：

- `GET /api/governance/dashboard`

展示内容：

- 任务状态汇总
- 输出申请状态汇总
- 输出文件与归档数量
- 规则包状态汇总
- 待审核结论数量
- 审计记录总量
- 最近任务
- 最近输出申请
- 最近输出文件
- 最近归档封存

## 5. 前端变化

### 5.1 规则包中心

规则包中心升级为树形规则编排器，支持：

- 嵌套分组
- 分组逻辑切换
- 规则条目增删
- 子分组增删
- 模板套用
- 条目插入

### 5.2 蓝白主题

全站主题由原暖色系改为蓝白治理风格：

- 页面背景改为浅蓝到白色渐变
- 主按钮、标签、边框统一为蓝色体系
- 面板与看板视觉更贴近政务/治理风格
