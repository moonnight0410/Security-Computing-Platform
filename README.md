# 政府部门联查数据计算单机系统

当前阶段：`Stage 12`

当前状态：规则包已支持直接导入登记，本域任务草稿已升级为完整核心模块，系统具备本域数据导入、规则包治理、任务执行、安全输出、归档验签和审计闭环能力。

## 项目简介

这是一个面向“所有数据不得出域”约束设计的单机本地系统。项目目标不是交换原始数据，而是在受控环境内完成：

- 本域数据导入与字段治理
- 规则包导入、验签、审批、修订与回溯
- 本域任务草稿创建、编辑、执行与结果联动
- 安全输出审批、归档封存、正式验签
- 审计留痕与治理看板

## 核心边界

- 原始数据不得出域
- 加密数据、哈希数据、去标识数据不得出域
- 派生主键、子键、对象级结果不得出域
- 仅允许输出执行回执、经审批结论声明、满足最小阈值的聚合统计

## 技术栈

- 后端：`FastAPI`
- 前端：`React 19`
- 构建：`Vite + TypeScript`
- 本地状态：`SQLite`
- 审计链：`JSONL + 链式哈希`

## 目录结构

```text
backend/        后端接口、模型、服务逻辑
frontend/       前端工作台、规则包编辑页、样式与类型
docs/           阶段报告、设计说明、验收文档
scripts/        启动、停止、校验、签名辅助脚本
workspace/      本地运行态目录
```

`workspace/` 主要包含：

```text
workspace/imports/    数据导入目录
workspace/exports/    输出文件目录
workspace/archives/   归档封存目录
workspace/audit/      审计链目录
workspace/logs/       本地日志目录
workspace/config/     本地状态库与密钥目录
```

## 已实现功能

### 本域数据与字段治理

- 本域 `CSV / XLSX / XLS` 导入
- 数据集画像生成
- 字段画像展示
- 主键、子键、敏感字段配置
- 聚合分组字段映射

### 规则包治理

- 规则包 `JSON` 导入登记
- 规则包单个验签
- 规则包单个审批
- 规则包批量验签
- 规则包批量审批
- 正式 `RSA` 公私钥验签
- 规则包编辑页
- 规则包草稿创建与修订保存
- 修订差异对比
- 规则包引用回溯
- 规则模板复用
- 规则条目复用
- 嵌套 `AND / OR` 组合规则编排

### 本域任务与执行

- 本域任务草稿创建
- 本域任务草稿编辑
- 本域任务草稿另存
- 本域任务草稿删除
- 本域任务草稿执行
- 任务结果摘要联动
- 输出策略控制：
  - `local_only`
  - `execution_receipt`
  - `manual_assertion`
  - `aggregate_summary`

### 输出治理

- 输出申请创建
- 输出申请审批
- 输出包预览
- 输出包落盘
- 输出文件归档封存
- 归档验签报告生成与复核

### 审计与治理看板

- 审计记录写入
- 审计链完整性校验
- 任务、规则包、输出、归档治理看板

## Stage 12 重点更新

- 工作台中的规则包登记不再手工填写“规则字段 / 规则操作符 / 规则值”，统一改为规则包文件导入。
- 新增规则包导入预检，导入前可查看名称、版本、签名人、签名引用、规则数和用途。
- 本域任务草稿升级为核心模块，支持创建、编辑、另存、删除、执行和结果联动查看。
- 后端新增任务更新与删除接口，并为任务对象补充 `updated_at`。
- 新增 Stage12 检查脚本：`scripts/run-stage12-checks.ps1`

## 启动方式

### 后端

```powershell
cd backend
python -m venv .venv
.\.venv\Scripts\pip install -r requirements.txt
.\.venv\Scripts\uvicorn app.main:app --host 127.0.0.1 --port 8000
```

或使用脚本：

```powershell
.\scripts\start-backend.ps1
```

### 前端

```powershell
cd frontend
npm install
npm run dev
```

或使用脚本：

```powershell
.\scripts\start-frontend.ps1
```

访问地址：

```text
http://127.0.0.1:5173
```

### 停止本地服务

```powershell
.\scripts\stop-local.ps1
```

## 检查脚本

当前推荐执行：

```powershell
.\scripts\run-stage12-checks.ps1
```

该脚本会执行：

- 后端 `compileall`
- 前端 `npm.cmd run build`
